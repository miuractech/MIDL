@miurac/admin

ADMIN - MIDL

This Repo contains modules and hooks that can be used to make an admin dashboard easily! This project uses Typescript but you can use Javascript if needed(or if you feel lazy).

All of this code belongs to Miurac.com. use of this code without permission is banned!

How to use

Copy the module you want to use from Midl folder. along with the lib folder. import the hooks and use it

import { useHook } from '../Midl/module';

const MyFunction = () => {
    const { hookFunctionality } = useHook();

}

It's as simple as that.
